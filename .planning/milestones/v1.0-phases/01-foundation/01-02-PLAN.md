<objective>
Build the Pall-Ex Nexus API client: the single-flight JWT token lifecycle manager (API-01), the circuit breaker with graceful fallback (API-05), and the MSW mock that lets every downstream surface run without credentials (API-06).

Purpose: This is the most delicate backend concern in the whole project (PITFALLS.md Pitfall 2). The token manager must guarantee that under concurrent calls, only one login/refresh fires — no 401 race cascade. The circuit breaker must ensure Nexus downtime degrades gracefully instead of crashing callers. The MSW mock must faithfully replicate the Nexus v2.2.1 response shape including edge cases (null ETA, not-found, 503). This plan is Wave 2 and runs in parallel with Plan 03 (database) — there is no file overlap.
Output: An authenticated, resilient, fully-mocked Nexus client exposing a single lookup function for the Tracking Service (Plan 04) to consume.
</objective>

Create `src/lib/nexus/circuit-breaker.ts`. Implement a minimal, dependency-free breaker (do NOT use opossum — see library_notes). Provide:

```typescript
export interface BreakerOptions {
  timeoutMs: number;            // default 5000
  errorThresholdPercentage: number; // default 50
  resetTimeoutMs: number;       // default 30000
  volumeThreshold: number;      // default 5
  now?: () => number;           // injectable clock for tests
}
export function createBreaker<A extends any[], R>(
  fn: (...args: A) => Promise<R>,
  fallback: (...args: A) => R,
  options?: Partial<BreakerOptions>,
): (...args: A) => Promise<R>;
```

State machine: `closed` -> (failures over volume+threshold) -> `open` -> (resetTimeout elapsed) -> `half-open` -> (success) -> `closed` / (failure) -> `open`. While `open`, return `fallback(...)` immediately without calling `fn`. Wrap `fn` in a timeout race: if it does not settle within `timeoutMs`, count a failure and return the fallback. Use the injectable `now()` clock (and accept a way to advance state in tests via fake timers) so tests are deterministic and fast.

The fallback must be a pure value-returning function — never re-throw — so callers (Plan 04) never see a raw Nexus error (PITFALLS.md Pitfall 9, success criterion 3).

Write `src/lib/nexus/circuit-breaker.test.ts` using vitest fake timers (`vi.useFakeTimers()`) to cover all five behaviors deterministically.
